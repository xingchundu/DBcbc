DDL 迁移（表结构 + 数据库对象）

从连接管理中选择源库与目标库，按两阶段执行迁移： ① 先迁移表结构（建表/补字段），② 再迁移数据库对象（存储过程、函数、触发器、包、视图、同义词、序列、Job、权限角色）。
支持路径：Oracle → PostgreSQL / 达梦 / MySQL、SQL Server → PostgreSQL。
